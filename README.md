# Fizzi — Premium Soda Landing Page

An interactive, high-energy landing page for **Fizzi**, a premium soda brand. The site delivers an immersive visual experience that reflects the brand's bold, vibrant personality.

## What is this site?

A **landing page** for a soda brand called **Fizzi**, showcasing its five flavors in an innovative, interactive way:
- Dynamic hero section with entrance animations
- Five flavor cards with hover interactions
- Rotating carousel for soda cans
- Parallax effects on scroll
- Dark / Light mode toggle
- Built-in language translation support

## Sections

1. **HeroSection** — Welcome area with entrance animations and interactive parallax
2. **FlavorsSection** — Five flavor cards with hover effects
3. **RotatingCanSection** — 3D rotating can with scroll-based parallax
4. **FlavorCarouselSection** — Interactive flavor carousel
5. **FeatureSection** — Detailed per-flavor feature displays
6. **MarqueeSection** — Auto-scrolling bottom marquee
7. **Footer** — Site footer

## Key Components

- **SodaCan** — Reusable soda can with animation support
- **BubbleBackground** — Animated floating bubble background
- **InitialLoader** — Interactive loading screen with spinning cans
- **MouseBubbleTrail** — Bubbles that follow the cursor
- **SiteEnergy** — Ambient site interaction effects
- **Navigation** — Responsive nav bar with dark mode support

## Getting Started

```bash
cd app
npm install        # Install dependencies
npm run dev        # Start dev server
npm run build      # Production build
npm run preview    # Preview production build
```
